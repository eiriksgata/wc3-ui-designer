# 插件系统使用说明

## 放置 lua54.exe

请将 `lua54.exe` 放在以下位置之一（按优先级排序）：

1. **打包后的可执行文件目录**（推荐）
   - 将 `lua54.exe` 放在与应用程序可执行文件相同的目录
   - 例如：如果应用是 `UI设计器.exe`，则将 `lua54.exe` 放在同一目录

2. **开发环境**
   - 将 `lua54.exe` 放在项目根目录或 `src-tauri` 目录

## 创建自定义插件

1. 在 `plugins` 目录下创建 `.lua` 文件
2. 插件必须返回一个函数，该函数接收两个参数：
   - `widgets`: 控件数组
   - `options`: 导出选项（包含 resourcePath, luaPath, mode, fileName 等）
3. 函数必须返回生成的 Lua 代码字符串

## 示例插件

参考 `plugins/example_plugin.lua` 文件，这是一个完整的示例插件。

## 使用插件

1. 在导出面板中选择"选择导出插件"
2. 点击"加载插件"选择你的插件文件
3. 或者点击"新建插件"创建新插件
4. 点击"编辑"可以修改现有插件
